## [Raidmode](https://github.com/federation-interservices-d-informatique/bot/blob/main/src/commands/moderation/raidmode.ts)
La commande `raidmode` active le raidmode, ce qui kickera tout les nouveaux membres pendant la durée du raidmode.

:::warning
Seul un membre du C.A de la FII peut faire ça.
:::

## Utilisation:
```
&raidmode
```
