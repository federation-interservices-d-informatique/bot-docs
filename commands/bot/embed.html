# Embed (opens new window)

La commande embed sert à créér un embed

WARNING

La permission ADMINISTRATEUR est requise pour utiliser cette commande

# Utilisation:

&embed {title: string} {description: string}

# Exemple:

Abysmal 01/03/2021
&embed Salut Ca va?
FII - Bot Bot 01/03/2021

Salut

Ca va?
